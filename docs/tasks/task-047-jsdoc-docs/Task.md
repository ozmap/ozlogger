# Task #47: Melhorar Documentação JSDoc

## Objetivo

Garantir que todas as classes, interfaces e métodos públicos tenham documentação JSDoc completa e precisa, melhorando o suporte a IDEs (IntelliSense).

## Descrição

Muitas funções core possuem apenas tipagem TypeScript, mas faltam descrições de parâmetros, exemplos de uso e anotações `@returns` / `@throws`.

## Critérios de Aceitação

- [ ] Cobrir 100% da API pública exportada
- [ ] Adicionar exemplos (`@example`) para métodos complexos
- [ ] Documentar opções de configuração (`LoggerOptions`)
- [ ] Revisar descrições de tipos utilitários
